# Snake Game
Snake Game Implementation Using C++ based on OO principles

#Code Structure
+ Singleton Class to handle the main two console jobs
+  --- Draw Point
+  --- Remove Point

+ Snake Class
+  --- List <Point> Class 

+ Point Class
+  --- Position X
+  --- Position Y

#Description
+ Simple implementation of Snake game using Console, Windows.h and C++
